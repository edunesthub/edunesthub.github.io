How to Install the Chawp App

1. Open

Visit app.chawp.me in your browser.

2. Add (iOS)

Tap Share, then Add to Home Screen.

2. Add (Android)

Tap Menu, then Add to Home screen.

3. Launch

Tap the Chawp icon on your homescreen to start using the Chawp app.
